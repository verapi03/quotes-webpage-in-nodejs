Node.js small project

This webpage is a project based on an exercise proposed by Marluan Espiritusanto from La Biblia de Node.js. To add a new quote to the webpage quotes you need to send a post request with Postman to the endpoint /quotes with a JSON like this: {"message":string, "author":string, "photo":stringURL}. The stringURL can be copied from any person's photo in https://www.ranker.com/crowdranked-list/the-most-influential-people-of-all-time
